## 📊 DATABASE SCHEMA

### Users Table
- id, email, password (hashed), role, timestamps

### Accounts Table
- account_id, user_id, account_number (unique)
- holder_name, phone, address
- balance (DECIMAL 12,2), account_status

### Transactions Table
- transaction_id, account_number
- transaction_type (DEPOSIT/WITHDRAWAL/INITIAL_DEPOSIT)
- amount, balance_after
- transaction_date, description

### OTP Table (Ready for future)
- otp_id, email, otp_code
- created_at, expires_at, is_verified

**Indexes on:**
- user email
- account_number
- user_id
- transaction account_number
- transaction date

## 🚀 QUICK SETUP (5 Minutes)

### 1. Database Setup (2 min)
```bash
mysql -u root -p < database_schema.sql
```

### 2. Configure Connection (1 min)
Edit: `DBConnection.java`
```java
DB_URL = "jdbc:mysql://localhost:3306/gift_bank"
DB_USER = "root"
DB_PASSWORD = "your_password"
```

### 3. Add Driver (1 min)
Place `mysql-connector-java-8.0.x.jar` in `WEB-INF/lib/`

### 4. Deploy (1 min)
Copy GiftBank to `TOMCAT_HOME/webapps/`

